## question two
Explain the difference between:
* Structured data
Structured data is a highly organised information that adheres a predefined model or schema typically arranged in rows and column within database. example; SQL and spreadsheets.
* Unstructured data.
Unstructured data refers to information that lacks a predefined data model or schema, can not stored, processed, or analyzed using a traditional rows-column database. examples; PDFs, images and emails.
